# Online-Greeting-Card-Editor
An Online Greeting Card Editor Using Fabric.js

Functions:
1. Select background template
2. Add Text
3. Modify Text
4. Delete Text
5. Drag the text box around on the canvas
6. Adjust text color
7. Download
